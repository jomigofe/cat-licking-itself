<h2>🛠️ Installation Steps:</h2>

<p>1. Clone rep</p>

```
git clone git@github.com:jomigofe/cat-licking-itself.git
```

<p>2. Go into folder</p>

```
cd cat-licking-itself
```

<p>3. Install dependencies</p>

```
npm install
```

<p>4. Run</p>

```
npm run dev
```
